# 传统中国风文化网页

以山水为骨，以水墨为魂，以诗词为文化底蕴。每一屏对应一首诗，每一次交互都是一次文化邂逅。

## 设计要点

- **视觉**：青花 #4A7C9B、朱砂 #C11B17、宣纸 #F8F4E9、墨色 #3E3A39；思源宋体/霞鹜文楷风格字体；最大宽度 1200px，边距 5%/30%。
- **动效**：滚动时背景渐变、卡片 3D 翻转（hover/点击锁定）、导航悬停水墨晕开、四季切换与 30 秒后樱花/雪花飘落。
- **结构**：顶栏（印章+站名+时令同赏）、Hero、四季区、文化卡片、诗词、墨迹画布、页脚。
- **加分**：TTS 朗读、毛笔墨迹画布、古琴音效提示、访问次数欢迎语、无障碍（aria、焦点样式、减少动效支持）、响应式（手机汉堡菜单、平板双列）。

## 如何运行

直接用浏览器打开 `index.html` 即可，无需服务器。若使用 VS Code，可安装 “Live Server” 扩展后右键 `index.html` 选择 “Open with Live Server”。

## 文件结构

```
网页设计2/
├── index.html      # 主页面
├── css/
│   └── style.css   # 样式
├── js/
│   └── main.js     # 交互与动效
├── assets/         # 可选：水印图、四季图、环境音
└── README.md
```

## 可选资源

- 在 `assets/` 中放入 `watermark.png` 可替换 Hero 区水印（在 CSS 中修改 `.hero-watermark` 的 background-image）。
- 环境音：在 `main.js` 中为 `ambientAudio.src` 指定音频路径即可播放。
